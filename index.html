Buttons

Use custom button styles for actions in forms, dialogs, and more with support for multiple colours and states.

Along with the default style, buttons come in 3 colors with a variation of that color:

  • Default
  • Lime, Lime Dark
  • Peach, Peach Dark
  • Slate, Slate Lite

Button Examples

Example 
<button class="button">Default</button>
<button class="button button--lime">Lime</button>
<button class="button button--lime-dark">Lime Dark</button>
<button class="button button--peach">Peach</button>
<button class="button button--peach-dark">Peach Dark</button>
<button class="button button--slate">Slate</button>
<button class="button button--slate-lite">Slate Lite</button>

Cards

Card Example

Example

    <span class="card__title">Card Title</span>
    <p class="card__content">
        Some quick example text to build on the card title and make up the bulk of the card's content.
    </p>
